### 📌 Build a React To-Do Checklist Component

Create a React component that displays a checklist of tasks and allows users to mark tasks as completed.

---

## 📋 Requirements

1. You are given an initial list of tasks:

* Each task should have:
  *  `id` (unique identifier)
  *  `label` (task name)
  *  `completed` (boolean)

1. Display all tasks with:

* A checkbox
* A label

1. When a user clicks on a checkbox:

* Toggle the `completed` state of that task

1. Display a summary:

* Show how many tasks are completed ( Counts )

---

## ⚙️ Functional Expectations

* Use **React functional components**
* Use **React Hooks (`useState`)**
* Update state **immutably**
* Ensure UI updates instantly when checkbox is toggled
* child components  props
