# 🧩 Star Rating for Cards

## 📌 Problem Statement

You are given a list of cards. Each card should have a  **star rating feature** .

---

## 📋 Requirements

* Each card should display **5 stars**
* User can click on a star to give a rating (1 to 5)
* Clicking a star should:
* Fill that star and all stars before it
* Each card should have its **own rating**
* Changing rating in one card should NOT affect others
